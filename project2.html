SOC-PULSE

Portable Offline Log Analysis Tool

SOC-PULSE is a lightweight, offline Security Operations (SOC) log analysis tool designed for isolated and air-gapped environments. It enables security teams to collect, analyze, and detect cyber threats locally without relying on cloud services or internet connectivity.

  • 100% Offline & Portable
  • Designed for air-gapped networks
  • Simple local web interface
  • Rule-based attack detection
  • Cross-platform (Win/Linux/Mac)
  • No external dependencies
